Job Portal Application

A full-stack MERN (MongoDB, Express, React, Node.js) platform designed to connect job seekers with recruiters.

🚀 Key Features

User Auth: Secure sign-up/login for students and recruiters using JWT.

Dashboard: Manage job postings, company profiles, and applications.

Responsive UI: Built with Tailwind CSS.

State Management: Uses Redux Toolkit for efficient data handling.

🛠 Tech Stack

Frontend: React, Tailwind CSS, Redux

Backend: Node.js, Express

Database: MongoDB

⚙️ Quick Start
Clone & Install: git clone npm install (in both frontend and backend folders).

Configure: Create a .env file in backend with MONGO_URI, PORT, and JWT_SECRET.

Run: Start the backend with npm run dev.
